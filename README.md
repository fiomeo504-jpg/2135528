# 🎲 Random Task Generator

**Автор:** Смоленцев Арсений Алексеевич

## 📌 О проекте

**Random Task Generator** — это консольное GUI-приложение на Python для генерации случайных задач. Программа помогает планировать день, чередуя учёбу, спорт и работу.

### Функционал:
- 🎲 Генерация случайной задачи из предопределённого списка
- 📝 Добавление своих задач с выбором типа
- 🔍 Фильтрация истории по типу (учёба/спорт/работа/все)
- 💾 Сохранение истории в JSON файл
- 📂 Автоматическая загрузка истории при запуске
- 🗑 Очистка истории с подтверждением
- 🧪 Встроенный тест всех функций
- 📊 Статистика по типам задач

## 🛠 Требования

- Python 3.7 или выше
- Стандартные библиотеки (tkinter, json, random, os)
