# Book Tracker — Трекер прочитанных книг

**Автор:** Смоленцев Арсений Алексеевич

## Описание программы

GUI-приложение для ведения списка прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц, а также фильтровать список по жанру и объёму.

### Функциональность:
- **Добавление книг** (название, автор, жанр, кол-во страниц)
- **Фильтрация** по жанру и количеству страниц (> заданного значения)
- **Просмотр** всех добавленных книг в таблице
- **Удаление** книг из списка
- **Сохранение и загрузка** данных в JSON-файл
- **Статистика** (общее количество книг и страниц)
